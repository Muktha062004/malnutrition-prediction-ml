# Medical Decision Support System for Malnutrition Prediction

This project predicts malnutrition in children under five years of age using machine learning.
It uses Bayesian Classification and K-Nearest Neighbors (KNN) to support early diagnosis
and healthcare decision-making.

## Technology Stack
- IDE: Visual Studio
- Programming Languages: C#, C++
- Back End: MS SQL Server
- Operating System: Windows 7 and above
